A simple webpage to prove the necesarry skills

If you want to run it and continue the webpage

	2. install node.js
	3. install gulp globally and then on project

	5. Have fun

	* Free stock images
	* HTML W3c valid
	* HTML5 Semantic ready
	* Modern and Clean
	* Google fonts
	* 100% responsive
	* HTML5, CSS3 & jQuery
	* Custom animations
	* Bootstrap 3
